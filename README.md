# moteus-motor-control

### Installation:
```
git clone https://github.com/pib-rocks/moteus-motor-control.git
cd  moteus-motor-control/python/balancing
pip install requirements.txt
python3 settings.py
```
### Balancing
```
python3 balancing.py
```
